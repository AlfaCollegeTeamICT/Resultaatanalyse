# Python resultaatanalyse

> Hoite Prins, Alfa-college Boumaboulevard

Python resultaatanalyse voor examenresultaten op basis van Eduarte BO Rapportage "DC030".

Script maakt grafieken van ingevoerde examenresultaten.

## Usage

1. Maak een map "input" aan
2. Plaats rapportage "DC030" in map input
3. Pas waar nodig bestandsnaam aan (standaard input.xlsx)
4. Voer `python3 main.py` uit
